Установка python
winget install Python.Python.3.11

Создание виртаульного окружения 3.11
py -3.11 -m venv venv_py311

Запуск
.\venv_py311\Scripts\activate   

Установка зависимостей
pip install -r .\requirements.txt

Настройка
Скопируйте env.example в .env и укажите PROXY_API_KEY

Запуск ассистента (из корня проекта)
python main.py

Альтернатива
cd assistant_api
python app.py
